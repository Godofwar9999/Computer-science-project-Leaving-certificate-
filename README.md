# Computer-science-project-Leaving-certificate-
Computer science project (Leaving certificate 2026)
The .hex file is to be inserted into a micro:bit, after which it will start functioning. It calculates the fire_score based on light intensity (in comparision to the sun) and temperature. Both can be shown and the device can be set to record data via pressing both button a and button b of the micro:bit simultaneously.
